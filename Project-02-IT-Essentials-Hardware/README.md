## تحليل القرارات (تبرير الاختيارات الفنية)

هذا هو "الدرس" الحقيقي: "ليش" تم اختيار كل قطعة لسيناريو "الموظف المتقدم".

### 1. المعالج (CPU)
* **القرار:** `Intel Core i5-14500`.
* **التبرير:** تم اختيار `i5` بدلاً من `i3` لضمان "قوة معالجة" (Processing Power) كافية. الموظف المتقدم (مثل المحاسب) يحتاج الأنوية (Cores) الإضافية لتشغيل ملفات الإكسل الثقيلة وبرامج التحليل بسرعة وبدون تعليق.

### 2. المذربورد (Mobo)
* **القرار:** `Gigabyte H610M S2H V2`.
* **التبرير:** تم اختيار "أرخص شريحة" (`H610`) تدعم المعالج (`LGA1700`) والرامات الحديثة (`DDR5`). هذا يحقق **"أفضل قيمة"** (Best Value)، حيث نوفر المال بدون التضحية بالمعايير الحديثة.

### 3. الرام (Memory)
* **القرار:** `16GB (2 x 8 GB)`.
* **التبرير:** `16GB` هو "المعيار الجديد" الضروري لـ (Windows 11 + برامج الأوفيس الثقيلة). الأهم، تم اختيارها كـ "قطعتين" (`2 x 8 GB`) لتفعيل تقنية **"Dual Channel"**، لضمان أعلى سرعة وأداء للرامات تحت الضغط.

### 4. كرت الشاشة (GPU)
* **القرار:** **لا يوجد كرت شاشة مخصص.**
* **التبرير:** هذا السيناريو (محاسب/محلل بيانات) لا يحتاج "معالجة رسومية" (Graphics) متقدمة. كرت الشاشة "المدمج" (Integrated Graphics) داخل معالج `i5-14500` كافٍ 100% لتشغيل الشاشات وأداء كل المهام المكتبية، وهذا يوفر مئات الدولارات في الميزانية.

### 5. الشاشة (Monitor)
* **القرار:** `24-inch 1080p`.
* **التبرير:** تم اختيار `24-inch 1080p` لأنه "المعيار الذهبي" (Sweet Spot) للشغل المكتبي، ويوفر **أفضل وضوح للنص (Text Clarity)** لراحة عين الموظف عند قراءة الأرقام والملفات لساعات طويلة.
