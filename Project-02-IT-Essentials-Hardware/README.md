## Decision Analysis — Why Each Part?

### 1. CPU — Intel Core i5-14500
Chosen over i3 to ensure sufficient processing power.
A power user (e.g. accountant) needs extra cores to run
heavy Excel files and analysis tools without slowdowns.

### 2. Motherboard — Gigabyte H610M S2H V2
Cheapest chipset (H610) that supports the LGA1700 socket
and DDR5 memory — best value without sacrificing modern
standards.

### 3. RAM — 16GB (2 x 8GB) DDR5
16GB is the new baseline for Windows 11 and heavy office
workloads. Chosen as a dual-stick kit specifically to enable
**Dual Channel mode** — maximizing memory bandwidth under load.

### 4. GPU — None (Integrated Graphics)
This scenario requires no dedicated GPU. The integrated
graphics inside the i5-14500 handles all display output
and office tasks at 100% — saving hundreds of dollars.

### 5. Monitor — 24" 1080p
The sweet spot for office work. 1080p on 24 inches delivers
the best text clarity for employees reading documents and
spreadsheets for long hours.
